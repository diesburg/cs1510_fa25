CS 1510 Python Information

Installing Python/IDLE on your computer

Python is free to install. If you have a PC or Mac and the ability to install programs, please install the newest version of Python from https://python.org. If you have Linux, come see me for the package names.

If you are renting a PC from UNI, please come see me, because we can often put in a request to get Python/IDLE installed.

Chromebooks and other situations

Some of you might be relying on a chromebook or a PC without the ability to install programs (like a borrowed or rented laptop). While not quite as good, you can use an online version of Python and IDLE to develop your programs. Here is an explanation video of how to get started.

If you have a chromebook and are comfortable using the command line, you can install Python and IDLE using the underlying Linux operating system. Here is an article that explains how.
